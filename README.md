# HR-employee-attrition-analysis
A complete HR analytics project using the IBM HR dataset from Kaggle. Data cleaning performed in Excel with inbuilt features, and interactive dashboards built in Power BI. Includes detailed documentation, KPI cards, slicers, and business insights to analyze employee attrition patterns and recommend strategic HR interventions.
